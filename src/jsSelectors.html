JavaScript Selectors

In JavaScript, selectors are used to select and manipulate HTML elements on a webpage. They are a crucial part of interacting with the Document Object Model (DOM) to dynamically update content, styles, and respond to user events. Here are the primary selectors in JavaScript:

1. `document.getElementById(id)`

  • Selects an element by its unique ID attribute.
  • Returns a single element with the specified ID.
  • Example: `document.getElementById("header")` selects the element with the ID "header".

2. `document.getElementsByClassName(className)`

  • Selects elements by their class name.
  • Returns a collection of elements with the specified class name.
  • Returns a live HTMLCollection (similar to an array but not exactly the same).
    • Example: `document.getElementsByClassName("container")` selects all elements with the class "container".

3. `document.getElementsByTagName(tagname)`

  • Selects all elements of a specific tag type (like div, p, etc.).
  • Returns a live HTMLCollection.
  • Example: document.getElementsByTagName('p') selects all

    elements.

4. `document.querySelector(selector)`

  • Selects the first element that matches a specified CSS selector.
  • Returns the first matching element or null if no matches are found.
  • Can use any CSS selector (like IDs, classes, or complex selectors).
  • Example: document.querySelector('.container') selects the first element with the class "container".

5. document.querySelectorAll(selector)

  • Selects all elements that match a specified CSS selector.
  • Returns a NodeList (similar to an array).
  • Can use any CSS selector.
  • Example: document.querySelectorAll('p') selects all

    elements.
